# LearnToGive

A tutoring platform that creates **threefold positive impact** through education.

## Our Mission

Founded in 2023 by passionate high school students who believe education can be a powerful force for change. We provide accessible academic support globally while funding scholarships for children in rural Thailand.

## How It Works

1. **Students** book affordable tutoring sessions (Â£10/hour) in subjects like Maths, Chemistry, Physics, Economics, and more
2. **100% of proceeds** go directly to scholarships for students in rural Thailand
3. **Thai scholars** receive 500 THB scholarships for essential school supplies

## The Threefold Impact

- **Tutees** gain knowledge, understanding, and new perspectives
- **Thai scholars** receive scholarships for school supplies (shoes, stationery)
- **Tutors** gain valuable teaching experience

## Features

- Online tutoring sessions via Zoom
- AI chatbot assistant (Adam) for instant help
- Live impact statistics from Google Sheets integration
- Subjects: Maths, English, Economics, Chemistry, Physics, Computer Science, SAT, IB, GCSE, A-Level

## Development

**Prerequisites:** Node.js

```bash
npm install
npm run dev
```

**Environment variables** (`.env.local`):
- `GEMINI_API_KEY` - For the AI chatbot
- `VITE_SHEETS_CSV_URL` - For live statistics from Google Sheets
